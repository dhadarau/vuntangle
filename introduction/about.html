# About

VUntangle library contains components and plugins intended to be used in Untangle application and products.

As a convention, an application using Vuntangle will be called further host application or host app.

The VUntangle components are based or extending the widely used vuetify (opens new window) UI library. Even so, the library does not includes the vuetify lib that will cause a large bundle, however it includes the agGrid (opens new window) package used by UGrid for data grids listings.

Good to know

Currently the host app have to include vuetify plugin so it contains all the vuetify components by default.

import Vue from "vue"
import Vuetify from "vuetify"
import "vuetify/dist/vuetify.min.css"

Vue.use(Vuetify);
export default new Vuetify({});
