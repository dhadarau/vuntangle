# Resources

# Main resources

Below are links to resources/libraries used in Untangle new UI applications/products.

  • VueJS (opens new window) - the base framework
  • Vue Router (opens new window) - the router to navigate routes/pages
  • Vuex (opens new window) - the state management pattern / library for data manipulation
  • Vuetify (opens new window) - the UI framework containing the UI elements from simple buttons to date pickers
  • Vue I18n (opens new window) - library adding localization support
  • VeeValidate (opens new window) - library used for form validation

INFO

To learn quickly and understand more you should check the documentation for each of the above references!

# Vue wrappers

  • agGrid (opens new window) - data grid used for table listings
  • highcharts (opens new window) - for graphs reports rendering

INFO

The above wrappers are current selection for grids and charting. This can be changed/replaced with whatever it might be more suitable for us.
